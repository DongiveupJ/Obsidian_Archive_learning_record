# 지금껏 학습한 것들을 문서화한 아카이브. 옵시디언을 사용해서 작성한다.

눈여겨 볼 포인트는, 시간이 경과할수록(3-1학기 -> 4-1학기) 마크다운 형식과 LLM을 활용한 지식 정리 방식이 향상되었다는 점 입니다.
예전의 문서(3-1/컴퓨터 네트워크, 병렬프로그래밍)는 직접 타이핑하면서 md형식을 제대로 활용하지 못했지만, 최근 문서(4-1/컴파일러, 컴퓨터 보안)는 수식이나 기호($...$ / ##, ==, ** 등)와 같은 형식을 제대로 활용하였으며, 향상된 LLM프롬프팅을 통한 요약 정리 자동화에 공을 들였습니다.
이를 통해 지식 정리와 문서화 능력이 경험이 쌓일수록 발전을 말씀드리고 싶습니다.
